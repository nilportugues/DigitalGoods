# DigitalGoods - Next.js project with redux & redux-saga

DigitalGoods is Next.js and Node.js based buy & sell platform. Allows creating a Progressive Web Apps.

Built with:
* next.js v8.1.0
* React v16
* Redux v4
* Now

## Installation
create .env file in the root folder
```
npm install
npm run start
```
